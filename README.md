# SimpleMaixMenu
A simple menu for maix amigo to use other scripts

HOW TO USE:
如何使用：

Upload this file to your tf card, and then create a dir named 'mods'. You can put your scripts in this dir.

把这个文件放在tf卡根目录然后建一个mods文件夹，把你的脚本比如例程放mods里就能选择使用了。
